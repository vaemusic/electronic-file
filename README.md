# DataV-vue3 demo

由于之前大佬写的 [DataV](http://datav.jiaminghi.com/) 不支持Vue3 Vite2.x，现部分代码用Vue3+TS重构。

本仓库是用重写后的[DataV-vue3](https://gitee.com/kjgl77/datav-vue3)实现，基础模板是[Anthony Fu](https://github.com/antfuhttps://gitee.com/kjgl77/datav-vue3)的[Vitesse-Lite](https://github.com/antfu/vitesse-lite)修改。
